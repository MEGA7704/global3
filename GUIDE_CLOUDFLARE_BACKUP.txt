
# GLOBAL3 — Cloudflare KV + D1 + Sauvegarde Cloud Réelle

## CONFIGURATION CLOUDFARE

### 1. Créer KV
Cloudflare → Storage & Databases → KV

Créer :
GLOBAL3_KV

Copier le ID.

---

### 2. Créer D1
Cloudflare → Storage & Databases → D1

Nom :
global3-database

Importer :
schema_global3_backup.sql

Copier le Database ID.

---

### 3. Modifier wrangler.json

Remplacer :

A_REMPLACER_PAR_VOTRE_ID_KV

et

A_REMPLACER_PAR_VOTRE_ID_D1

---

### 4. Déployer

Commande :

npm install
npx wrangler pages deploy public

---
